Portfolio

I have developed a number of interactive web applications that allow differing types of data to be visualised and manipulated. The data being visualised may be broadly classified into two categories - live, dynamically changing data and more static, fixed sources.

 

Dynamic Data Visualisation

 

Mayhem - The visualisation depicts the brain activity of participants whilst they receive external stimulus in the form of watching horror films.

Neurosis - The first brain controlled roller coaster ride. Here the brain activity of the participants is visualised whilst they enjoy the ride.

Storm in a Teacup - Actual wave data gathered off Hastings pier is used to derive a number of representations.

Oscilloscope - A data measuring tool used to validate the various data sources in use.

 

Data Visualisation Tools

 

VizTool - An application for visualising generic forms of multi-dimensional or time-varying data.

Sports data - Visualisation and analysis of sporting results, in this case football.

EnergyViz - Comparison of energy usage for cinema screenings over time.

TrajectASketch - A trajectory sketching tool.

 

Miscellaneous Tools

 

2D - Visualisation of 2-dimensional data, an alternative to 3D.

Interface - A generic visualisation interface to showcase particular objects.
